Заторник / Нагреватель

Создав оборудование и привязав к нему BNC-модуль нажимаем появившуюся кнопку Запуск

Перейдя на страницу созданного оборудования мы теперь можем управлять им. На этой страницы можно увидеть текущий режим работы и параметры самого модуля, а так же в сети модуль или нет, т.е. доступен для управления или нет.

Если параметры внутри модуля не соответствуют выбранному оборудованию, то необходимо в начале сбросить модуль на параметры по умолчанию нажав кнопку «Сброс»

Далее выбираем необходимый режим. Реализовано два режима работы:

Ручной — режим нагрева и поддержания заданной температуры. Этот режим удобно использовать например для нагрева промывочной воды. Для этого необходимо указать величину гистерезиса и целевую температуру и далее нажать кнопку «Старт». BNC-модуль начнет греть воду до указанной величины. После достижения заданной температуры начнется процесс поддержания этой температуры.

Автоматический — режим работы подходит для простейшего заторника. В этом режиме нужно указать величину гистерезиса и выбрать один из рецептов сохраненных в вашем профиле. Температурные паузы затирания в рецепте будут перенесены и нажав кнопку «Старт» BNC-модуль проведет по этим паузам отсчитывая время каждой паузы.

Управление BNC-модулем в данном режиме осуществляется полностью с данной страницы. При запуске программы BNC-модуль сохраняет настройки и при потери питания восстанавливает работу автоматически. Во время работы оборудования перезагружать страницу не желательно. Перезагрузка страницы при работе оборудования влияет на данные графиков, после перезагрузки график будет пустой.
