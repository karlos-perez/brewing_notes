Что такое BNC-модуль?

«BNC-модуль» - автоматика на основе умного Wi-Fi реле с термометром «Sonoff TH10 (TH16)».

Основная цель нашего проекта «BNC-модули» - дать пользователям (которые не имея особых навыков и бюджета) возможность создать себе на базе простого популярного и доступного Wi-Fi реле автоматику для нашего хобби (Пивоварение) с полной интеграцией с нашим сервисом рецептов. Что гораздо упрощает весь процесс создания пива: от расчётов рецепта до варки и сбраживания.

Что такое BNC-модуль по простому?

Это прошитый нашим программным обеспечением Sonoff TH10 (TH16) с термометром DS18B20.

Наше программное обеспечение позволяет объединить несколько модулей для работы над одной задачей. Таким образом можно сконфигурировать нужное оборудование из нескольких доступных модулей. Список возможного оборудования на данный момент можно посмотреть в разделе Виды оборудования.
